Proctopen

Bluetooth

  1. Press the mode button on the ESP32 until the OLED shows [Bluetooth].
  2. The ESP32's Proctopen WiFi hotspot will still be running for the Wemos units.
  3. Click Scan & Connect below and select Proctopen from the device picker.
Web Bluetooth requires Chrome or Edge. Does not work on Firefox or Safari.

WiFi — Access Point

  1. Press the mode button on the ESP32 until the OLED shows [WiFi AP].
  2. On this device, connect to WiFi network Proctopen with password proctopen123.
  3. Click Connect below.
Wemos units also connect to Proctopen automatically.

WiFi — Station

  1. Press the mode button on the ESP32 until the OLED shows [WiFi STA].
  2. First time? Join Proctopen WiFi and visit 192.168.4.1 to enter router credentials.
  3. Connect this device to the same router network, then click Connect.
Wemos units receive router credentials automatically from the ESP32.

USB Serial

  1. Press the mode button on the ESP32 until the OLED shows [USB Serial].
  2. Connect the ESP32 to this device via USB cable.
  3. Click Connect below. Your browser will show a port picker — select the ESP32 port.
Web Serial requires Chrome or Edge. Does not work on Firefox or Safari.
